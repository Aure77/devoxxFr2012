DevoxxFr2012
============

This application use Spring, Spring MVC, Spring Security, JPA 2

Prerequisite
------------

* Need a MySQL5 Instance + Datasource
* Need a JBoss AS 7

Usages
------ 

### Activate Local Profile (jee/jboss) :
    mvn clean package

### Activate CloudBees Profile (jee/jboss) :
    mvn clean package -Pcloudbees-soat
